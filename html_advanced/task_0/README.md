The html_advanced project focuses on building the complete structure of a webpage using only semantic HTML, without applying any CSS or visual styling. The main objective is to concentrate on organizing content logically and using the appropriate HTML tags.
It involves accurately translating a Figma design into clean and well-structured HTML code. This requires understanding the hierarchy of elements, respecting sections and headings, and following the overall layout defined in the original design.

Special attention must be given to best practices, especially in terms of accessibility. Using proper semantic tags improves code readability and ensures that the page is more accessible to assistive technologies such as screen readers.

Finally, this project aims to strengthen fundamental HTML skills by reproducing the structure of a professional design as precisely as possible. It helps develop discipline in code organization and provides a deeper understanding of how webpage content is structured before adding styles or interactivity.
